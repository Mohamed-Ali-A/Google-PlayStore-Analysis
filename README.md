# Google Play Store App Analysis

## Overview
Analysis of 8,196 Google Play Store apps to identify market entry 
opportunities for a startup entering the Android app market.

## Tools Used
- PySpark — data loading and cleaning
- Pandas — data transformation
- Tableau — visualizations and dashboard

## Key Findings
- Free apps average 10x more installs than paid apps
- FAMILY and GAME categories dominate total installs
- 80% of apps target Everyone content rating
- Smaller apps under 20MB consistently show higher install counts
